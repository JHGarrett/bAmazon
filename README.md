# bamazon
A program that uses MySQL and MySQL and Node.js.

bamazon is an Amazon-like storefront and will take orders from customers and update the stock
quantity from the store's inventory.  

## Technology Used
* NPM console.table
* NPM SQL
* NPM inquirer
* NPM dotenv
* Node.js
